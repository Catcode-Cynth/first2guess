# First2Guess 🎮

A fun real-time multiplayer guessing game where players compete to guess the secret first!

Built as a Web Development Assignment.

## ✨ Features

- Create and join game sessions using a 6-digit code
- Real-time multiplayer using Socket.io
- Game Master creates riddles, phrases, or secrets
- 60-second timer per round
- Each player gets 3 attempts to guess
- Winner gets +10 points
- Automatic Game Master rotation after each round
- Modern TikTok-style design with Tailwind CSS
- Confetti animation when someone wins
- Press **Enter** key to submit guesses quickly

## 🎯 How To Play

1. One player clicks **"Be Game Master"** to create a session
2. Other players join using the 6-digit code
3. Game Master creates a question/riddle and starts the round
4. First person to guess correctly wins the round
5. Game Master changes automatically after each round

## 🛠️ Technologies Used

- Node.js + Express
- Socket.io (Real-time communication)
- Tailwind CSS
- HTML & JavaScript
